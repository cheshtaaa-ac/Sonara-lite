# Sonara Lite

Sonara Lite is a Python-based hand tracking and gesture detection tool.  
It uses OpenCV and MediaPipe to detect fingers and recognize patterns in real time.

## Features
- Hand tracking using webcam
- Finger counting (0–5 fingers)
- Real-time gesture recognition
- Automatically stops the exercise session after reaching 20 exercises
- Hand counting still works after session ends
- Saves session stats to a JSON file

## Requirements
All dependencies are listed in requirements.txt.  
To install them, run:
pip install -r requirements.txt

## How to Run
1. Clone or download this repository.
2. Install dependencies.
3. Run the main script:
python "sent as official.py"
